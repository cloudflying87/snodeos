When you're ready to point a real domain at the site, update the server's .env file with these values and run ./build.sh:

DEBUG=False
ALLOWED_HOSTS=snodeos.com,www.snodeos.com
CSRF_TRUSTED_ORIGINS=https://snodeos.com,https://www.snodeos.com
SITE_URL=https://snodeos.com
DEFAULT_FROM_EMAIL=Brainerd Snodeos <noreply@snodeos.com>

If using a Cloudflare Tunnel, add CLOUDFLARE_TUNNEL_TOKEN=eyJh... as well. The tunnel handles HTTPS automatically — no certificate setup needed.

The site runs in Docker, so it can be hosted almost anywhere. Here are the most practical options for a club site:

Home or Office Server
Current setup
Run on any PC or mini PC on your home or office network. Use a Cloudflare Tunnel (already configured) to expose it to the internet without opening firewall ports.
  • Cost: $0/month (electricity only)
  • Hardware: Any machine that can run Docker (even a Raspberry Pi 4)
  • Uptime: Depends on your power/internet — fine for a club site
  • Cloudflare Tunnel: Free, handles HTTPS, hides your home IP
DigitalOcean Droplet
Recommended cloud option
A virtual server in the cloud. Run ./build.sh the same way. No Cloudflare Tunnel needed — the droplet has a public IP.
  • Cost: ~$6/month (Basic 1GB droplet)
  • Uptime: 99.99% SLA
  • Setup: Create droplet, install Docker, clone repo, run ./build.sh
  • Backups: $1.20/month extra for automated snapshots
Hetzner Cloud
European cloud provider with excellent price-to-performance. Same Docker setup as DigitalOcean.
  • Cost: ~$4–6/month (CX11 or CX21)
  • Uptime: 99.9% SLA
  • Data centers: US (Ashburn/Hillsboro) and Europe
  • Note: Choose US location for better latency from MN
Fly.io
Deploys Docker containers directly. No server management. Good free tier.
  • Cost: Free tier available; ~$5–10/month for persistent volumes
  • Setup: Install flyctl, run fly launch — done
  • Persistent storage: Fly Volumes for the media and database
  • Note: More DevOps knowledge needed for initial setup
Recommendation for a club: If someone is already running it at home and it's reliable, stay there — the cost is $0. If uptime matters (members checking trail conditions in season), a $6/month DigitalOcean droplet gives you cloud reliability with the same simple ./build.sh workflow.
