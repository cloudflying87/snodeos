Step 1 — Configure Email Required
Go to Email Settings

Email is used for password resets, application notifications, contact form alerts, dues reminders, and email blasts. It must be configured before any of those work.

Current state: Email is in "console" mode — nothing is actually sent. Follow the steps below to enable real email.
Option A — Gmail (Easiest)

Use a Google account dedicated to the club (e.g. brainerdsnodeos@gmail.com).

  1. Log into the club's Google account
  2. Go to Google Account → Security → 2-Step Verification and turn it on
  3. Go to Google Account → Security → App Passwords
  4. Create an App Password named "Snodeos Site" — save the 16-character password
  5. Add these to the server's .env file:
EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=brainerdsnodeos@gmail.com
EMAIL_HOST_PASSWORD=xxxx xxxx xxxx xxxx
DEFAULT_FROM_EMAIL=Brainerd Snodeos <brainerdsnodeos@gmail.com>
NOTIFICATION_EMAIL=brainerdsnodeos@gmail.com

Gmail allows up to ~500 emails/day — plenty for club use.

Option B — SendGrid (Better deliverability, 100 free/day)
  1. Create a free account at sendgrid.com
  2. Verify your sender domain (follow their DNS instructions)
  3. Create an API key under Settings → API Keys
  4. Add to .env:
EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=smtp.sendgrid.net
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=apikey
EMAIL_HOST_PASSWORD=SG.xxxxxxxxxxxxxxxxxx
DEFAULT_FROM_EMAIL=Brainerd Snodeos <noreply@yourdomain.com>
NOTIFICATION_EMAIL=officers@youremail.com
Step 2 — Text Messages (SMS) Optional

SMS is not yet built into the site but can be added. The most common service for this is Twilio. Once configured, it would enable text message dues reminders, meeting alerts, and trail condition notices.

Steps to Get SMS Working
  1. Create an account at twilio.com
  2. Purchase a US phone number from the Twilio Console
  3. Copy your Account SID and Auth Token from the dashboard
  4. Add to .env:
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_FROM_NUMBER=+12185550100
Once you have Twilio credentials, contact the developer. The dues reminder and blast pages can be updated to include a "Send as Text" option alongside the email option. The Member model already stores phone numbers when that field is enabled on the registration form.
Step 3 — Facebook Integration Optional
Go to Facebook Settings
Option 1 — Page Plugin

Embeds your Facebook page's recent posts on the home page — visitors can see your Facebook activity without leaving the site.

  1. Make sure you have a Facebook Page for the club (not a personal profile)
  2. Go to Settings → Facebook in this panel
  3. Choose "Facebook Page Plugin"
  4. Paste your Facebook Page URL
  5. Save — the feed appears on the home page
Option 2 — Zapier Auto-Post

When you post a Public announcement on the site, it automatically gets posted to your Facebook page.

  1. Create a free account at zapier.com
  2. New Zap: Webhooks → Catch Hook then Facebook Pages → Create Page Post
  3. Copy the webhook URL Zapier gives you
  4. Go to Settings → Facebook in this panel
  5. Choose "Zapier Auto-Post" and paste the URL
  6. Free Zapier plan: 100 runs/month — plenty for announcements
Step 4 — Domain & Go-Live Settings

When you're ready to point a real domain at the site, update the server's .env file with these values and rebuild:

DEBUG=False
ALLOWED_HOSTS=snodeos.com,www.snodeos.com
CSRF_TRUSTED_ORIGINS=https://snodeos.com,https://www.snodeos.com
SITE_URL=https://snodeos.com
DEFAULT_FROM_EMAIL=Brainerd Snodeos <noreply@snodeos.com>
If Using a Cloudflare Tunnel (home/office server)
  1. Log into cloudflare.com and add your domain
  2. Go to Zero Trust → Access → Tunnels and create a tunnel
  3. Copy the tunnel token and add to .env: CLOUDFLARE_TOKEN=eyJh...
  4. Rebuild with ./build.sh
