ASetting up Brevo (Recommended)
  1. Go to brevo.com and create a free account
  2. Verify your sender domain under Senders, Domains & Dedicated IPs
  3. Go to SMTP & API → SMTP tab → copy the SMTP key (starts with xsmtpib-) → paste above under Brevo SMTP Key
  4. For SMS: go to SMTP & API → API Keys tab → create a key (starts with xkeysib-) → paste above under Brevo API Key
  5. Fill in your Officer Alert Email and save
  6. Send a test email to confirm it's working
BSetting up Resend + Twilio
  1. Email: Create account at resend.com, verify your domain, create an API key (starts with re_) → paste above
  2. SMS: Create account at twilio.com, purchase a US phone number (~$1.15/month)
  3. Copy Account SID and Auth Token from the Twilio console dashboard
  4. Paste all three Twilio values above and save
  5. Send a test email to confirm Resend is working
