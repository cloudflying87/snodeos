Option A — Brevo (Email + SMS in one account)

Brevo handles both your club email AND text messages from the same account and API key. No separate Twilio account needed. Free tier includes 300 emails/day and SMS at ~$0.008/message.

Setup Steps
  1. Create a free account at brevo.com
  2. Go to Account → SMTP & API → API Keys and create a new key
  3. Add to the server's .env file:
# Brevo — handles BOTH email and SMS
EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
EMAIL_HOST=smtp-relay.brevo.com
EMAIL_PORT=587
EMAIL_USE_TLS=True
EMAIL_HOST_USER=your@email.com
EMAIL_HOST_PASSWORD=xsmtpib-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
DEFAULT_FROM_EMAIL=Brainerd Snodeos <noreply@snodeos.com>
NOTIFICATION_EMAIL=officers@youremail.com

# Brevo SMS (same account, different key)
BREVO_API_KEY=xkeysib-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
The EMAIL_HOST_PASSWORD for Brevo SMTP is your SMTP key (found under SMTP & API → SMTP). The BREVO_API_KEY is your API key (same page, different tab). They are different values.

After adding to .env, rebuild with ./build.sh and test from Email Settings. The Text Members page will unlock automatically.

Option B — Twilio (SMS only)

Twilio is the industry standard for SMS. More control over phone numbers and delivery. You would still need a separate email provider (Gmail, SendGrid, etc.).

  1. Create an account at twilio.com
  2. Buy a US phone number from the Twilio Console (~$1.15/month)
  3. Copy your Account SID and Auth Token from the dashboard
  4. Add to .env:
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_FROM_NUMBER=+12185550100
